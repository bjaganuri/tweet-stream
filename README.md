# tweet-stream

## Prerequisite to run application
### Install latest Nodejs LTS version
### Run node -v and npm -v commands from terminal to make sure that nodejs is installed properly

## Steps to run application
### 1. Open Nodejs termainal
### 2. Navigate to the root folder of application where it is located
### 3. Run npm run start-prod
### 4. Open http://localhost:3000 in browser
### 5.Observer that we page is loaded properly
### 6. Enter tweet search text in search box
